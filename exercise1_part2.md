git config
Wird am Anfang benutzt um das Benutzerprofil zu erstellen

git init
Damit wird der .git Ordner im derzeitigen Standort erstellt

git commit
Fügt alle durch git add hinzugefügten Dateien dem Ordner hinzu

git status
Zeigt an ob es noch nicht bestätigte Änderungen gibt

git add
Fügt eine Datei dem git repository hinzu

git log
Zeigt welche Kommandos, von wem und zu welcher Zeit diese ausgeführt wurden

git diff
Jede Zeile einer Datei wird vergleicht. Rote zeilen zeigen gelöschte oder veränderte Inhalte und grüne zeigt neu hinzugefügte Zeilen.

git pull
git pull holt die neuesten Änderungen von Github ab und fügt sie sofort auf dem lokalen Computer hinzu.

git push
Gegenteil von git pull. Git push bringen das Online Repository auf den Stand vom lokalen Computer.

git branch
Wird verwendet zum erstellen, Auflisten oder Verwalten von Zweigen im Repository.

git checkout
Wird benutzt um den Zweig zu wechseln.

git merge
Führt die Änderungen aus zwei verschiedenen Zweigen zusammen.


Am häufigsten wird git add und git commit benutzt.
